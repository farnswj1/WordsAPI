This is the Words API!

Have you ever wanted to obtain a complete list of English words? What about checking to see if something is even a word? Or check for words that have a particular sequence of letters? Then you're in luck! This API is a solution to these problems.

At the home page, the search bar allows you to search for a list of words that contain the input text. The results are then sent back as a JSON file, which the user can observe and see if there are any words that matched.

Or maybe you just want to use this for other programs. That's also fine! We provide API endpoints!

GET   /api   (retrieves the complete list of words)
GET   /api?search=[text]   (retrieves the words that match the specified text)
